# Locadora de Automóveis

## Como executar
1. Importar o projeto no IntelliJ ou Eclipse.
2. Rodar via Maven (`mvn clean install`) ou executar `TelaPrincipal.java`.

## Dependências
- Java 17+
- Maven
- Hibernate
- FlatLaf

## Telas
- Gestão de Usuários
- Gestão de Veículos
- Gestão de Aluguéis (com status)

## Uso de IA
Este projeto utilizou GPT-5 Mini para suporte na implementação de código.
